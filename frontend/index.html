// CAPABILITIES

Everything you need to parse technical documents

From multi-column blueprints to Excel datasheets — SpecStream extracts, orders, classifies, and exports structured data.

XY-Cut Partitioning
Recursive spatial division using projection profiles. Horizontal then vertical whitespace gaps are detected to split pages into logically-ordered reading zones.
Semantic Classification
Each block is tagged as heading, warning, footnote, table, or spec using font-size heuristics and pattern matching — no ML required.
Tree Hierarchy
A stack-based assembler reconstructs the document's logical parent-child structure from the ordered block sequence. Headings parent their subsequent content.
Excel & CSV Support
Upload .xlsx, .xls, .csv, or .tsv files. SheetJS parses them entirely in the browser with multi-sheet navigation, row inspection, and JSON export.
Interactive Inspector
Click any bounding box or tree node for bidirectional highlighting across the canvas, hierarchy tree, and reading-order list simultaneously.
JSON Export
Download the complete hierarchy as structured JSON with block type, label, bounding box coordinates, and page number — ready for LLM pipelines or downstream tooling.
